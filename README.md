# Student Analytics Project

This project is a beginner-level data analytics project using:

- Python
- Pandas
- NumPy
- Matplotlib
- SQLite Database

## Features

- CSV data analysis
- Database storage using SQLite
- Statistical analysis
- Data visualization
- Correlation analysis

## Technologies Used

- Pandas
- NumPy
- Matplotlib
- SQLite3

## Project Goal

To analyze how study habits and attendance affect student performance.
